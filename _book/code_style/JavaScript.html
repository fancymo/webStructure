JavaScript 规范

  1. 注释规范 /**
    • 这里的注释内容会被压缩工具压缩 */

/*！

  • 这里的注释内容不会被压缩工具压缩
  • 与上面一个注释块不同的是，第2个换成了! / 常常把文件的关键信息放在第2种注释内容里，如文件名称、版本号、作者等。

关键信息由关键词和 @key desc 格式书写:

关键词 描述 @auhor 作者 @param 参数 @example 示例 @link 链接 @namespace 命名空间 @requires 依赖模块 @return 返回值 @version 版本号
